1- Télécharger ce dossier
2- Ouvrez PHPmyAdmin et executer le code qu'il ya dans le fichier "BDDOrientation.sql"
3- Coller le dossier dans : C:\wamp64\www\
3- Allez sur un navigateur et allez à l'adresse URL : localhost
4- Dans "Outils" , cliquez sur "Ajouter une Virtual Host".
5- Ecrivez le nom de votre Virtual Host (myChoice par exemple)
6- Donnez le chemin du dossier : C:\wamp64\www\myChoice
7- Cliquez sur le boutton "Démarrer la création du VirtualHost"
8- Cliquez avec le boutton droit sur l'icone WampServer dans la barre de tache Windows > Outils > Redémarre du serveur DNS 
9- Vous pouvez maintenant accéder à votre site web en allant sur localhost > Vos VirtualHost ou en tapant l'adresse : localhost/myChoice
